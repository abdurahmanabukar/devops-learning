Challenge

Find the password for Level 1 stored in the `readme` file.
Solution
```bash
ls
ls -lah
cat readme
```
 Explanation

- `ls` lists files in the current directory.
- `ls -lah` shows detailed file information, including permissions and file sizes.
- `cat readme` displays the contents of the file.

 What I Learned
 I learned how to list files in a directory and read file contents using the `cat` command.
