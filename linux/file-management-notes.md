# Linux Setup Notes

## 5 Useful Commands I Learned

### 1. uname -a

Displays detailed system information, including the kernel version, operating system, and system architecture.

**Example:**

```bash
uname -a
```

Shows information about the current Linux system.

---

### 2. whoami

Displays the username of the currently logged-in user.

**Example:**

```bash
whoami
```

Shows the current user account.

---

### 3. pwd

Prints the full path of the current working directory.

**Example:**

```bash
pwd
```

Displays your current location in the file system.

---

### 4. ls -lah

Lists files and folders with detailed information, including permissions, owner, size, and hidden files.

**Example:**

```bash
ls -lah
```

Displays all files in the current directory.

---

### 5. hostnamectl

Displays the system hostname and operating system information.

**Example:**

```bash
hostnamectl
```

Shows details about the Linux system and hostname.

---

## What I Learned

I learned how to check my Linux system information, identify the current user, view my working directory, list files, and display system details.
